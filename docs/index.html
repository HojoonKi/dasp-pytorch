dasp

Differnetiable audio processors in PyTorch

Includes reverberation, distortion, dynamic range processing, equalization, stereo processing and more to come...

Enable applications in virtual analog modeling, blind parameter estimation, automated DSP, style transfer, and more.

Supports batching and operation on both CPU and GPU accelerators for fast training and reduced bottlenecks.

Open source and free to use for academic and commercial applications under the permissive Apache 2.0 licesnse.
